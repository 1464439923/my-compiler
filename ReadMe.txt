注意
--------------------------------------------------------------------------------------------------------------------------------------------
1. 可执行文件保存在.\compiler_v0\compiler_v0.exe。可执行文件依赖符号
表等文件，从该目录移动时需要连带这符号表等文件一起移动。
2. 编译器运行后，需要输入源文件相对或者绝对路径，对之进行编译解释执行。
3. 测试样例有
d.cpp （使用递归程序计算斐波那契数列第八个数的值）
e.cpp （生成1 22 333 4444字符串中间用0隔开）
z.cpp （计算给定层数的汉诺塔移动到另一个柱子的所需的次数）
其中有部分测试样例是给未加函数功能前的编译器运行的。
4. 压缩包中compiler_v0文件夹是visual studio 2015项目文件，打开文件夹
后使用visual studio 2015打开双击后缀为sln的解决方案文件，可以调试
